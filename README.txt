概要：
数据包含1157位棒球运动员的数据，包括他们的身高、体重、惯用手、击球率以及全垒得分。

可视化主要展现了全垒打次数与身高、惯用手以及击球率之间的关系。

设计：
通过对各变量之间的两两分析，发现身高与全垒得分之间有现对明显的正态分布，即有更高全垒打次数的球员的身高通常分布在中等水平故选择用身高（height）与全垒得分（HR）分别作为x轴与y轴做散点图以展现两个变量间的关系；

同时发现HR与击球率（avg）之间有正相关趋势，即随着击球率的升高，打出全垒打的次数也会缓慢增加，但不是特别明显，故设计散点的半径与avg正相关以体现差异；

通过对惯用手的分层，发现惯用手的不同对HR也会产生影响，简言之，惯用左手的球员平均打出全垒打的次数更多，惯用右手的球员次之，双手都用的球员平均打出全垒打的次数较低。故按惯用手设定散点的颜色；

作图发现，球员身高信息记录皆取整数，故使用Random功能是其在±0.5的范围内抖动，以更符合实际情况；

针对反馈1添加平均水准线、平均水准的鼠标悬停显示以及分层按钮方便读者比较；
针对反馈2，添加图例解释颜色以及半径代表的信息;
针对反馈4，调整数据抖动发生的时间，时其不会导致图形随意更改；添加平均水准线的图例，方便读者理解；修改y轴的解释性文字为HomeRun。
